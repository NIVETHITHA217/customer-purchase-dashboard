# Customer Purchase Behavior Dashboard

**Overview**
Excel-based interactive dashboard analyzing customer purchase behavior using a synthetic dataset.
Includes PivotTables, PivotCharts, slicers and a summary report.

**Files**
- `Customer_Purchase_Behavior_Dashboard.xlsx` — final interactive Excel dashboard
- `customer_purchase_behavior_dataset.csv` — cleaned dataset used in the project
- `report.pdf` — short project report with top insights
- `slides.pdf` — presentation slides
- `img/` — screenshots of the dashboard

**How to open**
1. Download `Customer_Purchase_Behavior_Dashboard.xlsx`.
2. Enable editing in Excel (if prompted).
3. Use slicers at top of the dashboard to filter by Year, Month, Gender, etc.

**Key Insights**
- Top categories: Electronics, Clothing
- Online channel drives majority of revenue
- Top customer segments: Age 25–35, Medium/High income
